Computing for Practical Statistics Link to heading
  • Lecturer/Supervisor: Dr Andres Benchimol

Predicting Brexit: A High-Dimensional Demographic Analysis

  • Handled high-dimensional data: addressed severe multicollinearity across 45+ socio-demographic variables by implementing Principal Component Analysis (PCA) and reduced the feature space into interpretable latent vectors while retaining more than 70% of the variance.
  • Utilized hierarchical clustering to engineer new categorical features that captured regional demographic similarities, further refining predictive power of the dataset.
  • Built and validated a Binomial Generalized Linear Model (GLM). Ensured the parameter interpretability and the model achieved a Pseudo R-squared of $0.85$. Delivered a robust and interpretable model that demonstrates the strong link between specific economic strata and political outcomes.
